# CataclysmBN_Mod_MachineBow
A mod by hamatk for the game Cataclysm Bright Nights. Adds a rare bow to the game that shoots bullets, complete with custom audio!

*Manual Installation instructions:*

The gun_bow folder goes in your game's mods folder.

For the ham_sounds folder...

A) If you already have a sound pack you like to use, then open the ham_sounds folder. Copy all the contents of that folder EXCEPT for soundpack.txt and paste them directly into your sound pack's folder.

B) If you DO NOT already have a sound pack you like to use, then take the entire ham_sounds folder and drop it into your game's sound folder.
